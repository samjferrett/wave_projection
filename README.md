# Wave Projection files

Files for the projection of q,v,r wave coefficients onto theoretical wave structures

## Files
1. Dn.nc - nc file containing parabolic cylinder functions as function of latitude, needed for wave_conversion.py.  
2. WESTPAC_Equatorial_Waves_dataset_updated.pdf - pdf file describing the qvr files and the projection process.
3. wave_conversion.py - python script for conversion of q,v,r files to u,z,v by projection onto wave structures.
